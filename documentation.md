## Make a script executable
```sh
# Set permissions
chmod +x [filename]

# Run
sh [filename]
```

## Refresh symlinks 
```zsh
sh ~/dotfiles/scripts/refresh-symlinks.sh
```

## Update and cleanup system
```zsh
update # it's a alias ;)
# or
sh ~/dotfiles/scripts/clean-ubuntu.sh
```
